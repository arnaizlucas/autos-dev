# 1 Clonar el repositorio
git clone https://github.com/arnaizlucas/autos-dev.git

# 2 Entrar a la carpeta del frontend
cd autos-dev

# 3 Instalar dependencias
npm install

# 4 Iniciar el servidor
npm start

La app se abrirá automáticamente en http://localhost:3000

Se utilizo React + Material UI. 
Se agregaron filtros dinamicos con Select y Slider para mostrar componentes diferentes, al igual que estilos de los botones.
Se implemento una modularizacion acorde de los componentes para poder aplicar cambios rapidamente.
Se configuro el backend para poder mostrar algunos datos.
